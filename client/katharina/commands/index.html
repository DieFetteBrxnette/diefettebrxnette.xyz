Music
Command Description
k!join / k!play Makes the bot join your voice-channel
k!play [song/playlist] Adds the song to the queue
k!queue Shows the queue
k!track Provides information about the current song
k!skip Skips the current song
k!pause Pauses/Unpauses the music
k!loop Loops the queue or only the current song
k!shuffle Mixes the queue
k!bassboost Choose how much the bass should be boosted
k!stop Stops the music
k!lyrics Get the lyrics of the current song
Reaction-Commands
Command Description
k!ticket Starts the setup process for the Ticket-Manager
k!ticket Shows the currently open tickets, if it's already setup
k!ticket setup Starts another setup process to make more Ticket-Managers
k!ticket add [user] Adds a user to the ticket
k!ticket remove [user] Removes a user from the ticket
k!ticket close Closes a ticket
Invite-Manager
Command Description
k!invitemanager Manage the settings of your Invite-Manager
k!bonus add [user] Gives a user a bonus invite
k!bonus remove [user] Removes a bonus invite from a user
k!bonus clear [user] Clears all bonus invites from a user
Moderation
Command Description
k!check Checks for missing permissions
k!clear [amount] Deletes the given amount of messages
k!kick [user] [reason] Kicks a user from the server
k!ban [user] [reason] Bans a user from the server
k!mute [user] [duration] [reason] Mutes a user from the chat
k!warn [user] [reason] Gives a user a warning
k!warns [user] Shows the warnings of a certain user
k!softban [user] [duration] [reason] Bans a user softly from the server (loses access to all channels, but stays on the server)
